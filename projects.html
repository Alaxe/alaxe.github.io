Gemini

A co-op platformer in which 2 players have to solve puzzles made out of logic gates in order to traverse the level. To play together they need to join the same room (no registration is required). Phaser and javascript are used for the client, Node.js for the server and communication is done through WebSockets. The game was developed for the 2017 Softuniada where it won 3rd place (age 16 and over).

JudgeSystem

A web-based system for grading solutions to algorithmic problems. I. e. You submit your code (C/C++), it gets compiled, run on some unit tests and you get feedback on which tests have passed successfully. It supports time limits, memory limits, and custom output checkers. The project uses the Django backend framework, the Celery task queue and Bootstrap CSS. The project was awarded 4th place at the 2016 SoftUniada (age under 16).
Github

Stealth

A 2D stealth game written in javascript. The player has to steal some coins from the level, while not being spotted by the guards. What was most interesting in this project was developing an efficient algorithm for the lighting and detection by the guards. The game has been made using the Phaser framework and the Tiled level editor.
Website Github

ChochoRun

A simple runner with procedural level generation. The player controls a panda which has to collect small bamboo sticks while avoiding big ones. The game gets progeressively faster and at some point the player "dies" and gets a score. This project is written in Java using Libgdx - a framework which lets you deploy your game on multiple platforms.
Website Github

BetweenWorlds

A game where you have to run through a maze, but with a twist - there are two "worlds" (or layers), connected by portals. To beat the level the player has to constantly switch between the worlds and even activate some of the portals. Each time you face a bigger procedurally generated maze. The game was made in 48 hours for the 30th Ludum dare game jam using Libgdx.
